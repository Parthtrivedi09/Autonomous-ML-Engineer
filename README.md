# Autonomous ML Engineer

An AI-powered autonomous machine learning system that analyzes datasets, performs exploratory data analysis, engineers features, selects models, trains and evaluates them, and generates detailed reports using LLM-based agents.

## Features (Planned)

- Dataset profiling
- Automated EDA
- Feature engineering
- Intelligent model selection
- Hyperparameter tuning
- Explainable AI (SHAP)
- Retrieval-Augmented Generation (RAG)
- PDF report generation
- Streamlit interface

## Tech Stack

- Python
- LangChain
- LangGraph
- Groq (Llama 3.3 70B)
- Hugging Face Embeddings
- FAISS
- Scikit-learn
- Pandas
- Matplotlib
